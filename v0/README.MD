# Salty-crackers (v0)

`v0` is the first playable version of **salty-crackers**, a simple 2D platform game built with plain HTML, CSS, and JavaScript.

## Game Overview

You play as a pregnant woman collecting salty crackers (`galletitas saladas`) across platform levels.

- The game has **9 levels**.
- Each level requires collecting all crackers to advance.
- At each new level, the character's womb/belly is visually larger.
- After finishing level 9, a final scene shows the baby being born.

## Controls

- Desktop:
  - `Arrow Left`: Move left
  - `Arrow Right`: Move right
  - `Arrow Up`: Jump
- Phone:
  - `Left` button: Move left
  - `Right` button: Move right
  - `Jump` button: Jump

## Gameplay Rules

- Collect every cracker in the level.
- Avoid falling off platforms (you respawn at the start area).
- Once all crackers are collected, the game transitions to the next level.

## Tools and Technologies Used

- **HTML5**: Game page structure and canvas element
- **CSS3**: Layout and visual styling
- **JavaScript (Vanilla)**: Game loop, physics, collisions, input handling, level progression, rendering
- **Canvas API**: Drawing player, platforms, collectibles, and final scene
